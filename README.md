# Agente-de-IA-para-Mantenimiento-Preventivo-de-Flota
El agente opera bajo una política estándar de mantenimiento preventivo común en la industria logística.

## Agente de IA para Mantenimiento Preventivo de Flota

Este proyecto es un prototipo conceptual desarrollado para el Hackatón Traxión Tech Challenge.
Simula el funcionamiento de un agente de IA que evalúa el riesgo de mantenimiento de unidades
de transporte con base en una política estándar (20,000 km o 6 meses).

### El sistema incluye:
- Interfaz web en HTML/CSS
- Formulario de entrada de datos operativos
- Resultado simulado del diagnóstico del agente

#### Próximos pasos:
- Integración con API de IA
- Conexión con telemetría y ERP de mantenimiento
- Dashboard en tiempo real
